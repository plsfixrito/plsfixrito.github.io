Determines whether a unit will collide with a circular missile along a predetermined path. Circular missiles cause collision only at the destination point.

Parameters
unitThe unit to check collision.
missileStartPosThe starting point of the missile.
missileEndPosThe destination point of the missile.
missileSpeedMissile's speed.
missileRadiusMissile's collision radius.
delayThe time (in milliseconds) it will take for the missile to spawn.
extraRadiusThe extra hitbox radius you can assign to the unit. Default value: 0.

Determines whether two moving objects will collide within a predetermined path. The movement is considered linear.

Parameters
startThe starting point of the first object.
destinationThe destination point of the first object.
hitboxThe hitbox radius of the first object.
speedThe movement speed of the first object.
isUnitDetermines whether the object is a unit. Set to true if the first object disappears after reaching its destination.
start2The starting point of the second object.
destination2The destination point of the second object.
hitbox2The hitbox radius of the second object.
speed2The movement speed of the second object.
isUnit2Determines whether the object is a unit. Set to true if the second object disappears after reaching its destination.
